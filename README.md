# Асинхронный веб-краулер

Веб-краулер для парсинга сайтов на Python (asyncio + aiohttp).

Разработка ведётся инкрементально в ветках по дням — каждая следующая
ветка продолжает предыдущую (`day-1` → `day-2` → … → `day-7`):

- `day-1` — базовый асинхронный HTTP-клиент (aiohttp, таймауты, пул соединений)
- `day-2` — парсинг HTML и извлечение данных (BeautifulSoup + lxml)
- `day-3` — очередь обхода, управление конкурентностью, фильтрация URL
- `day-4` — rate limiting и соблюдение robots.txt
- `day-5` — классификация ошибок, повторы с backoff, circuit breaker
- `day-6` — сохранение результатов: JSON Lines, CSV, SQLite
- `day-7` — sitemap, статистика и отчёты, конфигурация, CLI, Docker

Полное описание проекта, инструкция по запуску и документация — в README
ветки `day-7`.
